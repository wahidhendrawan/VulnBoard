VulnBoard

A self-hosted vulnerability management platform for VAPT consultants — generate professional pentest reports, manage engagements, and import scanner results across multiple security frameworks.

View on GitHub Quick Start
ℹ️ This is a documentation page, not a live demo. VulnBoard requires a backend (Express + Prisma + SQLite) with authentication and persistent storage, so it can't run as a static site. Run it locally with Docker in under a minute — see below.

Features

Multi-Framework Reports

OWASP Top 10 2021, NIST CSF 2.0, ISO 27001:2022, PCI DSS 4.0, SANS/CWE Top 25.

Scanner Import

Import findings directly from Burp Suite XML, OWASP ZAP JSON, and Nmap XML.

PDF Export

One-click professional PDF reports via headless Chromium (Puppeteer).

Findings Library

40+ reusable finding templates in English and Bahasa Indonesia.

White-label Reports

Per-engagement logo and brand color for client-ready deliverables.

Authenticated Workspaces

JWT + bcrypt auth with per-consultant persistent engagements.

Run it locally

git clone https://github.com/wahidhendrawan/VulnBoard.git
cd VulnBoard
cp .env.example .env   # set a strong JWT_SECRET
docker compose up -d

Frontend on :5173, backend API on :4000. Seed demo data with docker compose exec backend npm run db:seed.
